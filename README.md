# Diabetes Risk Prediction & Health Insights System

## Overview
This project predicts the likelihood of diabetes using machine learning and provides insights into key health factors affecting risk.

## Features
- Data analysis and visualization
- Logistic Regression model
- Confusion matrix evaluation
- Streamlit web app for real-time prediction

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn
- Streamlit

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the app:
   streamlit run app/app.py
